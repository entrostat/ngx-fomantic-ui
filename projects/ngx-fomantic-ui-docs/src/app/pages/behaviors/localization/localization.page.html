Localization

Localization lets you control what language components display in

Usage

The localization service allows you to choose the language that the components are displayed in. Note that these settings only affect components with pre-defined text, such as the datepicker.

To change locale, inject the FuiLocalizationService service into your root component, as well as importing a locale from those provided by the library.

All components that display any non customizable text will now display in Spanish.
